Clone the repository.

```sh
git clone https://github.com/Matheus13F/AluraQuiz-NextJs.git
```

`cd` into the directory.

```sh
cd AluraQuiz-NextJs
```

Install the project dependencies:

```sh
yarn

# or

npm install
```

Start the development server:

```sh
yarn dev

# or

npm dev
```
